## Digital submission

### Format

PDF/A (archival quality PDF).

The `ctu-thesis build` command generates standard PDF. Check with your department whether PDF/A conversion is needed.

### File naming convention

```
[StudentID]_[LastName]_Thesis_[Year].pdf
```

Example: `B2026001_Doe_Thesis_2026.pdf`

### File size

Maximum 50 MB. Use `ctu-thesis build --draft` during development and the full build only for final submission.

## Submission checklist

- [ ] Cover page with correct information
- [ ] All front matter sections complete
- [ ] Page numbering correct (Roman → Arabic)
- [ ] All figures and tables numbered and captioned
- [ ] All references cited in text
- [ ] References in IEEE format
- [ ] Minimum page count met (50+ pages)
- [ ] Abstract within word limit (200-350 words)
- [ ] 3-5 keywords provided
- [ ] Acknowledgements included
- [ ] Advisor signature on evaluation page
- [ ] PDF file size under 50 MB
- [ ] Printed copies properly bound

## Timeline

### Typical 6-month schedule

| Month | Milestone | Deliverable |
|-------|-----------|-------------|
| 1 | Topic selection and proposal | Proposal document (5-10 pages) |
| 2 | Literature review | Chapter 1 draft |
| 3-4 | Design and implementation | Chapters 2-3 draft |

2-3 weeks before defense.

| Phase | Duration |
|-------|----------|
| Presentation | 15-20 minutes |

## Common mistakes to avoid

### Formatting

| Wrong | Correct |
|-------|---------|
| Arial or Calibri font | Times New Roman 13pt |
| Left margin 3cm | Left margin 4cm (for binding) |
| Double spacing (2.0) | 1.2 line spacing for main text |
| No first-line indent | 1cm first-line indent |

### Content

| Wrong | Correct |
|-------|---------|
| Abstract > 400 words | Abstract 200-350 words |
| Only 2 keywords | 3-5 keywords |
| < 15 references | Minimum 15 references, preferably 20+ |
| All references from websites | Mix of journals, books, conferences |

### Citations

| Wrong | Correct |
|-------|---------|
| APA or MLA style | IEEE style only |
| Unreferenced figures/tables | All figures/tables referenced in text |
| Missing figure captions | All figures have descriptive captions |
